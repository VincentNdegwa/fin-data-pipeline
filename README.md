# Project: Financial Transactions Cleaning

This project contains scripts and notebooks for cleaning and analyzing financial transaction data.

## Structure
- data/raw: Raw, unprocessed data files
- data/processed: Cleaned and processed data files
- notebooks: Jupyter notebooks for exploration and analysis
- scripts: Python scripts for data processing

## Setup
1. Create a virtual environment: `python -m venv venv`
2. Activate the environment and install dependencies: `pip install -r requirements.txt`

## Usage
- Run scripts in the `scripts/` directory for data processing.
- Use notebooks in the `notebooks/` directory for interactive analysis.
